Software and organizations using or contributing to PROJ¶

Free and open source software¶

  • GDAL Translator library for raster and vector geospatial data formats, used by > 100 other software

  • GRASS GIS A raster/vector open source GIS

  • libgeotiff Library to read/write GeoTIFF keys

  • MapServer A popular web mapping application

  • PDAL Point Cloud Data Abstraction Library

  • PostGIS Spatial database extender for PostgreSQL

  • QGIS A cross platform desktop GIS.

  • QField Mobile data collection app for Android, iOS and Windows.

  • R A free software environment for statistical computing and graphics

  • mundipy A Python framework for spatial data analysis

PROJ bindings in other languages¶

See Language bindings

Proprietary licensed software¶

  • FME A GIS translator package. Includes a PROJ based transformer.

  • Avenza Maps Top mobile mapping app for offline navigation and data collection.

  • Bunting Labs No-code GIS infrastructure

Geodetic organizations¶

  • Agency for Data Supply and Infrastructure of Denmark (SDFI)

  • Land Information New Zealand (LINZ)

  • National Land Survey of Finland (NLS)

  • National Land Survey of Iceland (NLS)

  • Netherlands Partnership Geodetic Infrastructure (NSGI)

    • Netherlands’ Cadastre, Land Registry and Mapping Agency (Kadaster)

    • Directorate-General for Public Works and Water Management (Rijkswaterstaat)

    • Hydrographic Service of the Royal Netherlands Navy

  • Nordic Geodetic commission (NKG)

  • Norwegian Mapping Authority (Kartverket)

  • Swedish mapping, cadastral and land registration authority (Lantmäteriet)

  • USGS: PROJ initial author, Gerald E. Evenden, worked at USGS
